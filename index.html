📅 Fecha límite para pronósticos
Cuando llegue esta fecha/hora, el formulario se bloquea automáticamente y nadie podrá cargar más pronósticos. Déjalo vacío para no tener límite.
⚽ Forzar solapa Pronósticos
Cuando ya hay eliminatorias disponibles, la solapa "Pronósticos" se oculta sola (la fase de grupos terminó). Actívala solo si quieres que siga visible igual, para revisarla.
⏱️ Fechas tope eliminatorias (fallback)
Normalmente el cierre de carga de cada fase se calcula solo: 12 hs antes del primer partido de esa fase, según la API. Si la API no funciona, fija aquí manualmente el cierre. Déjalo vacío para usar el cálculo automático.
📣 Avisos del día
Cartel de bienvenida que aparece al entrar, solo el día de la fecha indicada, durante todo ese día. Marca "Activo" para habilitar cada aviso y desmarca para dejar de mostrarlo (si no está marcado, no se muestra). Sirve para un "¿Sabías que...?", "¡Último momento!", etc.

⬇️ Exportar pronósticos

🌎 Equipos y jugadores
Mantén aquí la lista de selecciones y sus jugadores (sin tocar el código). Elige un equipo para editarlo, o "Nuevo equipo" para agregar. Los jugadores van uno por línea. Estos datos alimentan los selectores de mejores/peores/campeón y el buscador de goleadores.
🥅 Goles por jugador
